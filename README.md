# AdventureWorks Sales & Customer Intelligence Dashboard

## Project Overview

This project analyzes sales performance, customer behavior, and product performance using SQL Server and Power BI.

## Tools Used

* SQL Server
* SSMS
* Power BI
* Excel

## Key Metrics

* Total Revenue: 123M+
* Total Orders: 31K+
* Total Customers: 19K+
* Total Products: 500+

## Dashboards

### Executive Dashboard

* Revenue Analysis
* Territory Analysis
* Monthly Revenue Trend

### Customer Analytics

* Top Customers
* Customer Segmentation

### Product Analytics

* Top Products
* Product Performance Analysis

## Key Insights

* Bikes generated 86% of total revenue.
* Southwest was the highest-performing territory.
* Mountain-200 products generated the highest revenue.
* Revenue grew significantly between 2011 and 2013.
